## Descripción del proyecto

Este proyecto implementa un sistema básico de gestión para un restaurante utilizando Programación Orientada a Objetos en Python.

El sistema permite registrar y listar productos, bebidas y clientes mediante un menú interactivo ejecutado desde consola.

La aplicación aplica principios SOLID para mantener una estructura modular, organizada y fácil de mantener.
